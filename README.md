## What is this?
this is python version management tool.



## install
```
$ chmod +x install.sh
$ ./install.sh
```

## command
- pyenv install -l
  - show list of python version you can install.

- pyenv install xxx(3.7.0)
  - install python.

- pyenv global xxx(3.7.0)
  - global mode.

- pyenv local xxx(3.7.0)
  - local mode.

- pyenv versions
  - show list of installed python version.
